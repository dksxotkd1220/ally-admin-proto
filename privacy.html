개인정보처리방침

공지일자: 2026년 07월 24일 시행일자: 2026년 07월 24일

주식회사 인시퀀스(이하 "회사")는 A11Y 글로벌 서비스(이하 "서비스")를 운영하며, 개인정보 보호법 등 관련 법령을 준수합니다. 이 방침은 서비스 이용 과정에서 수집되는 개인정보를 어떻게 처리하는지 안내합니다.

01

개인정보의 수집 및 이용 목적

회사는 다음의 목적을 위해 개인정보를 수집·이용합니다.

  • API 키 발급 및 계정 식별 (이메일이 곧 계정입니다. 별도 회원가입은 없습니다)
  • 이메일 인증 링크(매직 링크) 발송
  • 발급된 API 키와 워드프레스 사이트 연결·관리
  • 서비스 공지 및 주요 변경사항 안내
  • 출시 알림 및 신규 플랫폼 지원 소식 안내 (별도 동의자에 한함)
  • 서비스 개선을 위한 통계 분석

02

수집하는 개인정보 항목 및 보유 기간

수집 목적 수집 항목 보유 기간
API 키 발급·계정 관리 이메일 주소, 워드프레스 사이트 URL 삭제 요청 시까지
출시 알림 신청 이메일 주소, 기대 기능 선택값 (선택) 수신 거부 또는 삭제 요청 시까지
제품 업데이트·마케팅 수신 이메일 주소 수신 거부 또는 삭제 요청 시까지
서비스 이용 기록 (자동 수집) IP 주소, 접속 일시, 브라우저 정보 1년

얼리 액세스 기간 동안 결제 정보는 수집하지 않습니다. 정식 요금제 출시 시 별도로 동의를 받아 처리합니다.

03

만 14세 미만 아동의 개인정보

서비스는 만 14세 미만 아동(일부 국가 기준 만 16세 미만)을 대상으로 하지 않습니다. 아동의 개인정보가 수집된 사실을 인지한 경우 해당 정보를 즉시 삭제합니다.

04

개인정보의 제3자 제공

회사는 원칙적으로 이용자의 개인정보를 제3자에게 제공하지 않습니다. 다만, 다음의 경우는 예외입니다.

  • 이용자가 사전에 동의한 경우
  • 법령에 따른 수사기관의 적법한 요청이 있는 경우 (영장 기재 범위 내에 한함)

05

개인정보 처리 위탁

회사는 서비스 운영을 위해 다음 업체에 개인정보 처리 일부를 위탁합니다.

수탁업체위탁 업무보유 기간
Oracle Cloud 클라우드 인프라 운영 (서울 리전) 위탁 계약 종료 시까지

회사는 위탁 계약 시 개인정보 보호법에 따른 필요 사항을 명시하고 수탁자를 감독합니다.

06

개인정보의 파기

보유 기간이 경과하거나 처리 목적이 달성된 경우 개인정보를 지체 없이 파기합니다. 전자적 파일은 복구 불가능한 방식으로 영구 삭제하며, 종이 문서는 분쇄 또는 소각합니다. 법령에 따라 보존이 필요한 경우 별도로 분리하여 보관합니다.

07

정보주체의 권리와 행사 방법

이용자는 언제든지 다음의 권리를 행사할 수 있습니다.

  • 개인정보 열람 및 사본 요청
  • 잘못된 정보의 정정 요청
  • 개인정보 삭제(계정 및 API 키 삭제) 요청
  • 마케팅 수신 동의 철회
  • 개인정보 처리 정지 요청
요청 방법
이메일 a11y@inseq.co.kr 로 문의해 주세요. 요청 접수 후 10영업일 이내에 처리합니다.

08

개인정보의 안전성 확보 조치

  • 개인정보 및 API 키는 암호화하여 저장·관리합니다
  • 이메일 전송 구간은 TLS 암호화를 적용합니다
  • 접근 권한을 담당자로 최소화하고 정기적으로 점검합니다
  • 접속 기록을 보관하고 위변조를 방지합니다
  • 보안 프로그램을 설치하고 주기적으로 갱신·점검합니다

09

쿠키 및 자동 수집 정보

서비스는 이용 편의 및 보안을 위해 쿠키와 유사 기술을 사용할 수 있습니다. 브라우저 설정에서 쿠키를 거부할 수 있으나, 일부 기능(매직 링크 인증 등) 이용이 제한될 수 있습니다.

접속 IP, 접속 일시, 브라우저 종류 등 서비스 이용 기록은 보안 및 오류 분석 목적으로 자동 수집됩니다.
