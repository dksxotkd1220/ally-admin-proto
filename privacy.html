우리가 수집하는 정보

A11Y.so는 서비스 제공에 필요한 최소한의 정보만 수집합니다.

정보수집 시점이유
이메일 주소 API 키 발급 신청 시 계정 식별 및 인증 링크 발송. 이메일이 곧 계정이며, 별도 비밀번호는 없습니다.
워드프레스 사이트 URL API 키 발급 신청 시 발급된 API 키를 특정 사이트에 연결하고 무단 사용을 방지하기 위해 사용합니다.
이메일 주소 출시 알림 신청 시 새 플랫폼 지원 등 서비스 소식 발송. 언제든 수신 거부할 수 있습니다.
접속 기록 (IP, 브라우저, 접속 일시) 서비스 이용 중 자동 수집 보안, 오류 분석, 서비스 개선. 1년 후 자동 삭제됩니다.
얼리 액세스 기간에는 결제 정보를 수집하지 않습니다. 정식 요금제 출시 시 별도로 안내하고 동의를 받아 처리합니다.

수집한 정보의 사용 방식

  • API 키를 발급하고 해당 사이트에 연결·관리하기 위해 사용합니다.
  • 이메일 인증 링크(매직 링크)를 발송하기 위해 사용합니다.
  • 서비스 공지, 중요 변경사항, 보안 알림을 발송하기 위해 사용합니다.
  • 동의한 경우에 한해 제품 업데이트·접근성 관련 소식을 보냅니다.
  • 서비스 품질 개선을 위한 익명화된 통계 분석에 활용합니다.

정보를 공유하는 경우

우리는 귀하의 정보를 팔지 않습니다. 아래의 경우에만 제한적으로 공유합니다.

  • 인프라 파트너: 서버 운영을 위해 Oracle Cloud(서울 리전)를 사용합니다. 귀하의 데이터는 Oracle의 인프라에 저장되며, Oracle은 당사의 지시에 따라서만 데이터를 처리합니다.
  • 법적 의무: 법원 명령, 수사 영장 등 적법한 법령 요청이 있는 경우 요청 범위 내에서만 제공합니다.

위 두 가지 외에는 어떤 경우에도 귀하의 개인정보를 제3자와 공유하지 않습니다.

데이터 보관 기간

우리는 서비스 제공에 필요한 기간 동안만 정보를 보관합니다.

  • API 키 계정 정보(이메일·사이트 URL)는 삭제 요청 시 즉시 파기합니다.
  • 마케팅·출시 알림 수신자 정보는 수신 거부 요청 시 즉시 제거합니다.
  • 자동 수집된 접속 기록은 1년 후 자동 삭제됩니다.

귀하의 권리

귀하는 언제든지 아래 권리를 행사할 수 있습니다.

  • 열람: 우리가 보유한 귀하의 정보를 확인할 수 있습니다.
  • 수정: 잘못된 정보를 정정할 수 있습니다.
  • 삭제: 계정과 관련 데이터 전체를 삭제 요청할 수 있습니다.
  • 처리 제한: 특정 목적의 데이터 처리를 중단 요청할 수 있습니다.
  • 수신 거부: 마케팅 이메일은 수신 거부 링크 또는 이메일로 언제든 중단할 수 있습니다.
  • 이의 제기: 거주 지역의 개인정보 보호 감독기관에 불만을 신고할 수 있습니다.

보안

우리는 귀하의 정보를 보호하기 위해 다음 조치를 취합니다.

  • 이메일 전송과 API 통신은 TLS 암호화를 적용합니다.
  • API 키와 개인정보는 암호화하여 저장합니다.
  • 내부 접근 권한을 필요 최소한으로 제한하고 정기적으로 검토합니다.
  • 보안 취약점을 발견하셨다면 a11y@inseq.co.kr 로 알려주세요.
