자주 묻는 질문

도입 전에 궁금한 것들.

API 키는 어디서 발급받나요?
이 페이지에서 이메일만 입력하면 바로 발급됩니다. 회원가입은 필요 없고, 입력하신 이메일이 곧 계정이 돼요. 이메일당 키는 하나만 발급되며, 잃어버려도 같은 이메일로 다시 확인 링크를 받을 수 있습니다.
적용하려면 개발이 꼭 필요한가요?
아니요. 워드프레스 플러그인을 설치하고 발급받은 API 키만 입력하면 작동합니다. 서버 설정이나 기존 소스코드 수정이 필요 없어요. 모든 설정은 워드프레스 관리자 화면에서 끝납니다.
적용하면 얼마나 걸리나요?
플러그인 설치와 API 키 입력은 약 3분. 새로 추가되는 이미지 또는 기존 이미지를 일괄적으로 AI가 대체 텍스트를 생성할 수 있어요.
사이트 전체에 적용되나요, 일부만 고를 수 있나요?
둘 다 됩니다. 전체 일괄 적용도, 원하는 페이지·이미지만 골라 적용도 가능합니다. 워드프레스 설정에서 대상 페이지나 이미지를 선택하고 대체 텍스트를 생성할 수 있어요.
새로 만드는 사이트에도 되나요?
네. 신규 구축 사이트에도 플러그인만 설치하면 바로 쓸 수 있고, 개발 단계에서 미리 설명을 만들어 두면 오픈 시점부터 접근성을 갖춘 채 시작할 수 있어요.
도입 후에도 계속 관리되나요?
네. 새 이미지가 올라오면 자동으로 감지해 대체 텍스트를 생성할 수 있는 옵션이 있어요. 한 번 깔고 끝나는 도구가 아니에요.
인증을 대신 받아 주나요?
아니요. 앨리는 인증을 대행하지 않습니다. 대신 접근성 점검에서 가장 많이 지적되는 항목인 이미지 대체 텍스트를 기준에 맞게 채우는 일을 자동화합니다. (어떤 도구도 소송이나 인증을 100% 보장하지는 않습니다.)
